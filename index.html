Què fem?

Fem tecnologia web

Un projecte web o mòbil és la suma de molts elements: estratèngia, integracions, etc... Aquests elements no poden estar limitats per la tecnologia. A Iskra fem tècnicament possible allò que tens al cap.

Fem software web

Seleccionem o desenvolupem el programari que més s'adapti a les necessitats de cada projecte. Volem que els gestors encarregats del projecte web o mòbil, siguin totalment independents a l'hora de gestionar el contingut del projecte. Creem projectes que ajudin a fer la vida més fàcil.

Fem Open Source

Som i creiem en el software de codi obert. En som desenvolupadors, usuaris i defensors. Creiem en les comunitats que treballen en projectes comuns per fer productes i projectes de qualitat i en benefici de tothom.

Fem back-end

Desenvolupem backends en python. Fent-los a mida i escalables. Des de Guillotina, Django o Plone. Dockeritzem.

Fem front-end

Desenvolupem frontals moderns amb tecnologies de frontal modernes, com per exemple React o Angular. Fem lògiques de frontal.

Fem Machine Learning

Fem machine learning aplicat a la productivitat de l'empresa.

Stack tecnològic
